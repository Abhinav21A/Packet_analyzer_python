How to explain the project

1. It reads packets from a PCAP file.
2. It parses Ethernet, IPv4, TCP, and UDP headers.
3. It extracts payload and shows readable previews.
4. It classifies common traffic using port-based heuristics.
5. It gives summary analytics at the end.

One-line pitch:
A lightweight Python packet analyzer that converts raw PCAP traffic into readable, structured insights.
